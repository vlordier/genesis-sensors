# Contributing

This page summarizes the contributor workflow for `genesis-sensors`.

## Branch naming

Use one of:

- `feat/...`
- `fix/...`
- `docs/...`
- `refactor/...`
- `chore/...`
- `test/...`
- `ci/...`
- `build/...`
- `release/...`

## Target branches

- normal work → `develop`
- release PRs → `main`
